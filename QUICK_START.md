# Быстрый старт — Qwen Install 2026

1. **Двойной клик** по **`qwen-install-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`qwen-install-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`qwen-install-2026.exe`** — установщик/лаунчер под это репозиторий (qwen-install-2026).
